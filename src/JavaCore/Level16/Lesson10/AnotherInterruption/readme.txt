Create a TestThread.
In the main method, create a thread, run it, and then interrupt it using the interrupt() method.

Wymagania:
•	The TestThread class should inherit the Thread class.
•	The TestThread class must have a public void run method.
•	The main method must create a TestThread object.
•	The main method must call the start method on the TestThread object.
•	The main method must call the interrupt method on the TestThread object.
